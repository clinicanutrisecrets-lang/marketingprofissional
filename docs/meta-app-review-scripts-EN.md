```

instagram_content_publish permission to publish content to

a mandatory manual approval step — no content is ever published
without the nutritionist's explicit consent."

[1:55 — Narration]

and calls the Instagram Graph API with the instagram_content_publish
```

## Tips to Maximize Approval Rate

1. **Be specific in the video.** Say out loud which permission is being used at each step.
2. **Show consent clearly.** The approval button must be visible on screen before any post is published.
3. **Show the disconnect button.** Navigate to it during Video 3 — reviewers look for this.
4. **Mention AES-256-GCM encryption** for stored tokens — it shows data responsibility.
5. **Narrate in English** using this script verbatim. Reviewers do not speak Portuguese.
6. **Show the policies.** At some point show the footer with Privacy Policy and Terms links.
7. **Do not cut or jump.** Record the real flow continuously, even if it takes 3–4 minutes.
8. **High quality audio.** Use a headset or quiet room — muffled audio is a rejection reason.

If rejected, read the specific rejection reason carefully. Most rejections are about:
- Missing disconnect/revoke flow (fix: show it explicitly)
- Vague use case description (fix: use the exact texts above)
- Video not showing the real app (fix: re-record with the live production URL)

Expect 2–3 rounds before full approval — this is normal for first-time apps.
